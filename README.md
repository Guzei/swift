# Операторы
```
+ сложение
- вычитание
* умножение
/ деление
% остатот от целочисленного деления
= присваивание
+= увеличение себя
-= уменьшение себя
*= умножение себя
/= деление себя
условие ? да : нет    -- тернарный
== сравнение "равно"
!= сравнение "не равно"
>, >=, <, <=
!переменная    -- отрицание
переменная!    -- жёсткое извлечение опционала
..., ...<    -- диапазон
тип?   -- расширение типа до опционала
??  -- мягкое извлечение опционала
```
